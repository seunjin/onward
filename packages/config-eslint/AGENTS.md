# ESLint 설정 패키지

## 범위

- 모노레포에서 공용으로 쓰는 ESLint flat config를 담당한다.

## 주의사항

- 앱별 예외 규칙을 과하게 넣기보다 작고 예측 가능한 preset을 선호한다.
- 여기 변경은 워크스페이스 전체에 영향을 줄 수 있으므로 보수적으로 다룬다.
- 특정 앱 하나를 위해 모든 패키지에 부담이 되는 규칙을 넣지 않는다.

## 작업 규칙

- 앱과 패키지 전반에서 재사용 가능한 설정을 유지한다.
- Node와 React처럼 실행 환경 차이는 preset으로 분리한다.
- 새 규칙을 넣을 때는 false positive 가능성을 먼저 점검한다.

## 변경 후 확인

- 최소 루트에서 `pnpm lint`를 실행한다.
